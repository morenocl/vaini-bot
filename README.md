### Como ejecutar?


Editar el archivo `.env` con el valor del token de tu bot y el nombre de tu app heroku. Luego ejecutar:

~~~
$ virtualenv env
$ source env/bin/activate
$ pip install -r requirements.txt
$ python main.py
~~~

`/info`   Muestra el estado de la lista.
